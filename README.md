### ⚙ Installation Guide

1️⃣ Clone Repository
git clone https://github.com/your-username/react-from-scratch.git

2️⃣ Navigate to Project
cd react-from-scratch

3️⃣ Install Dependencies
npm install

4️⃣ Start Development Server
npm run dev

Open browser:

http://localhost:5173/
